# Agent Working Rules
When working on this project, always follow the rules below and complete specific steps at appropriate times/scenarios:

## Timing: Before Task Starts
If **LEARNING/index.md** exists, read **LEARNING/index.md** to learn from experience and knowledge.

## Timing: Implementing Features/Fixing Bugs/Refactoring Code
Invoke the **bdd-atdd** skill to complete code modification tasks based on behavior and acceptance criteria.

## Timing: After All Code Modifications Are Complete
Invoke the **doc-update** skill to execute documentation updates.
